How to use

Simply copy and paste the id of any video into the text box, press the button or hit enter, and the video should change. Click the player to start the video and then hit either left or right and then immediately either up or down; this should start the game. When you die you should be able to just hit up or down to start the game again.

How to get the video ID

In the address bar, the id is between 'v=' and '&' if any. For example, with a url of youtube.com/watch?v=QW54WsjrxPY&index=1&list=RDQW54WsjrxPY the id is QW54WsjrxPY. Another example, if the url is youtube.com/watch?v=xKyGcAiLkDQ then the id is xKyGcAiLkDQ.
